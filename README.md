# PassLounge — NCLEX Prep by Student Nurse Lounge

AI-powered spaced repetition flashcard app for NCLEX-RN preparation.

## Live App
https://passlounge.vercel.app

## Tech Stack
- **Current:** Single-file HTML/CSS/JavaScript + Supabase
- **Refactor (in progress):** React 18 + TypeScript + Vite + Zustand

## Environments
| Environment | URL | Supabase Project |
|---|---|---|
| Production | passlounge.vercel.app | passlounge-production |
| Staging | staging.passlounge.vercel.app | passlounge-staging |
| Dev | localhost:5173 | passlounge-dev |

## Repository Structure
- `docs/` — Product framework and architecture decisions
- `src/` — React app source (TypeScript, in progress)
- `supabase/migrations/` — Database schema history
- `index.html` — Current production app (live until React refactor completes)
